## java-assignment

### Project Packages:

#### algorithms: 
Contains the implementation of the algorithm exercise.
Includes one class with the implementation of the solution.

#### design:
Contains the solutions to the design questions.
Contains two text files, one for each solution.

#### main and model:
Contains the task management system
   main - The Main class.
   model - All the classes.
   
### Run Instructions   

- Download the project: git clone https://github.com/rut26376/java-assignment.git

#### Run the Task Management project
- Update the JSON file path inside model.TaskRepository according to your local path.
- run the class main.Main

#### Run the algorithmic exercise
- run this class: algorithms.StrictlyIncreasing
